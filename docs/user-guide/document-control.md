## Overview

Documents in Cascadia PLM support:
- File attachments and version control
- Linking to Parts and other items
- Check-out/check-in workflow
- Multiple file formats

## Creating a Document

1. Navigate to **Documents** in the sidebar
2. Click **New Document**
3. Fill in the required fields
4. Upload file attachments
5. Click **Save**

## File Management

- **Check Out**: Lock a document for editing
- **Check In**: Upload new version and release lock
- **Download**: Get the latest file version
- **History**: View all file versions
